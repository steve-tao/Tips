# Install matplotlib on GhostBSD 19.04

## Admin tasks
1. Install pkgconf and freetype.
```bash
    pkg install pkgconf
    pkg install freetype
```
## User tasks
1. Install matplotlib.
```bash
    pip install --user matplotlib
```
